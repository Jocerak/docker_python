# My Python App

A simple Python app using Flask.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/my-python-app.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   python app.py
   ```
